1º Aula 

As redes de computadores: 
são um conjunto de componentes que servem para troca de informação. Seu objetivo principal é o tráfego de dados entre 
computadores, por meio de ferramentas de hardware e software, utilizando protocolos.

Internet: 
é um sistema que conecta computadores do mundo inteiro por meio de redes 
interligadas.

Surgimento da Internet:
contexto da guerra fria e corrida espacial, EUA constrói as ARPANET (1957), que seria
uma rede de transmissão de dados locais (inicialmente poucos estados, 
porteriormente quase EUA todo). O Projeto ARPANET possibilitou o 
compartilhamento de  informações online. A ARPANET era uma rede fechada. Esse 
projeto foi o precursor da internet.

3º Aula 

Infraestrura da Internet
- Internet é uma rede de computadores que interconecta dispositivos
por todo mundo
- Interliga sistemas finais (hospedeiros) como: notebooks, tablets, 
celulares, TVs, jogos, etc
- Os sistemas finais são conectados entre si por meio de links de comunicação
e comutadores de pacotes
- Há vários tipos de enlaces (links de comunicação) como: cabos coaxiais,
fios de cobre, fibras óticas, ondas de rádio. Os diferentes tipos de link
transmitem dados com diferentes velocidades ou taxas, dadas em bits por segundo.

Funcionamento da transmissão de dados via INTERNET
- Envios de dados de um sistema final para outro: sistema final remetente -> sistema
final receptor 
- os dados são adicionados a bytes, formando os pacotes;
- os pacotes são enviados por um comutador (roteadores) ao seu enlace de comunicação
- a sequência que o pacote percorre por enlaces e comutadores, é chamada de rota ou caminho

Protocolos de Rede
- são regras que regem os processos de comunicação 
- Transport Control Protocol (TCP) é um dos principais protocolos e opera na 
camada de transporte, acrescentando funcionalidades, como garantia do fluxo de dados
na transmissão.
- Internet Protocol (IP) opera na camada de rede e possibilidade comunicação.
- uma porta TCP está associada, de forma direta, a uma aplicação
ou a um serviço específico, que utilizam protocolos específicos.

HTML
- linguagem de marcação de hipertexto - tag language;
- não é uma linguagem de programação - não há uma compilação de um programa executável
- o código é responsável por exibir na tela o que foi codificado em HTML;
- arquivos resultandos do html são pequenos;
- não necessita de um editor especial.

linguagens que funcionam com HTML:
- JavaScript e VBScript (eventos) / PHP (banco de dados) / JAVA (interfaces gráficas)

Um documento HTML é dividido em seções: início, cabeçalho, título, corpo e fim.

Navegação da Internet:
- Estrutural
é uma navegação em que uma página da WEB está ligada a outra página, de acordo
com uma hierarquia;

- Associativa
tem relação com links de hipertexto: como rodapés, quando você pausa o mouse em um link e 
aparece as relações daquele link, por exemplo;

- Utilitária 
por exemplo, uma página que apresenta opções, como escolher o idioma de uma página.


Tipos de página da Web:
- Navegacionais: 
direciona os visitantes ao conteúdo procurado, páginas principais, por exemplo;

- Conteúdo: 
artigos, textos, notícias, videos, etc - são páginas que contém motivos para pessoas visitarem o site;

- Funcionais:
possibilitam realizam de ações como, por exemplo: buscas, compras, verificação de emails, consultas, cadastros, etc.

Links e Navegação Web:

- links são textos ou elementos gráficos que conduzem o usuário para outras páginas ou

3º Aula 

Sites informativos: apresentam informação desejada para o usuário de um modo de fácil acesso.

Sites interativos: caracterizados por terem interfaces funcionais, possuem grande capacidade de processamento de dados e programação avançada.

Fases na criação de um site:

1: briefing - ideia geral do site; 
2: orçamento ao cliente;
3: aprovação do orçamento;
4: elaboração do esboço do site (protótipo);
5: envio do esboço ao cliente;
6: aprovação do esboço pelo cliente;
7: desenvolvimento do deign do layout do site;
8: desenvolvimento efetivo do site;
9: realização de testes e SEO (técnicas que influenciam os algoritmos dos buscadores);
10: definição das ferramentas usadas para a administração e estratégias de marketing; 
11: aprovação final do site.

Design de navegação: amplitude e profundidade

amplitude: número de itens de menus existentes em uma página

profundidade: número de níveis hierárquicos existentes na estrutura

quanto maior amplitude, menor a profundidade 

quanto menor amplitude, maior a profundidade


Características desejáveis para um site:
- facilidade de utilização;
- layout atraente - gerar conforto ao usuário;
- clareza visual;
- desempenho e velocidade do site.

Publicação de páginas web e sites:
- páginas agrupadas de modo coerente;
- design coerente em termos de estrutura, identidade visual, fonte, cores, etc;
- ter identidade visual;
- registrar um nome de domínio 

Tipos de hospedagem:
- compartilhada: sites compartilhando o mesmo servidos e seus recursos, como: memória, processamento,
etc. Indicado para pequenas empresas e blogs pessoais.
- servidor privado virtual: aloca um espaço reservado de processamento, memória e espaço em disco
em um servidor. Indicado para empresas de médio porte.
- nuvem: A nuvem hospeda recursos de computação de uma rede de servidores virtuais e físicos, permitindo maior
escalonabilidade e flexibilidade para fazer alterações rapidamente.

Segurança na Web:
- a segurança cibernética é o esforço contínuo para proteger esses sistemas em rede 
e todos os dados de usos não autorizados ou prejudiciais.

Vulnerabilidade:
- confidencialidade: restrição e respeito à privacidade das informações do indivíduo;
- integridade: prevenção contra alterações ou destruições não autorizadas de informações;
- disponibilidade: a informação deve estar disponível para o usuário - representa confiabilidade.

Conceitos de ataque e ameaça:
ataque: tentativa de violação;
ameaça: possibilidade de haver um ataque;

Tipos de ataque: 
falsificação de e-mail, ataque de força bruta (tentativa e erro de usuário e senha), desconfiguração de página (defacement), scam (se passar por outra pessoa), farsas, etc.


4º Aula 

Violações:
- de sigilo: duplicação de informações
- de integridade:modificação de dados
- de disponibilidade: destruição de dados 
- furto de serviço: aproveitamento de algum recurso de forma não autorizada
- recusa de serviço: impedir que o sistema continue operando ou ferindo sua funcionalidade

Tipos de ataques:
- DoS: intenção que o serviço tenha sua execução interrompida;
- backdoors: acesso oculto de programas;
- wiretapping: interceptação de chamada de voz;
- web inaceitável: utilização de forma não permitida pela organização;
- advance fee fraud: conduzir uma pessoa de fornecer informações confidenciais;
- phising: tenta obter dados pessoais de um usuário através de um site que parece confiável.

Malwares: junção de malicious e software
exemplos: malwares infecciosos (vírus e worms); malwares não infecciosos (cavalo de troia);

Vírus: 
- vírus é um programa ou parte de um programa, normalmente malicioso, que se propaga pela
intrudução dele mesmo, faz várias cópias de si;
- trata-se de um malware que normalmente da execução ou de um arquivo hospedeiro para se tornar ativo.

há vários tipos de virus: 
- sorrateiros: ocultos ao usuário - infectam o disco rígido e executam processos sem consentimento.
- dorminhocos: permanecem inativos durante a maiior parte do ano e agem em datas específicas.

Worms ou vermes:
- é um programa capaz de se multiplicar por uma rede;
- semelhante ao vírus, mas não se propagam por cópias de si em outros arquivos, mas pela execução direta das suas cópias;
- consome muitos recursos computacionais.

Cavalo de troia:
- usa da aparência para enganar o usuário;
- uma vez que é executado, segue instruções de ataque autorizados pelo usuário.
